## Commands to run this project

npm i
php artisan migrate

## To setup Inertia.Js
composer require inertiajs/inertia-laravel
php artisan inertia:middleware
composer require tightenco/ziggy

php artisan db:seed


## To run project
php artisan serve
npm run dev
npm run tailwind

goto the route http://127.0.0.1:8000/customers
